Workshop: Building and Deploying Multi-Agent Systems with ADK and Vertex AI

This hands-on workshop teaches participants how to build multi-agent AI systems using Google's Agent Development Kit (ADK) and Vertex AI Agent Engine. Attendees will learn how to create and orchestrate AI agents powered by Gemini models, build custom tools, and implement sequential, iterative, and parallel workflows. The workshop covers design pattern trade-offs between LLM-driven and workflow-driven architectures, agent coordination strategies (Sub-agents vs. Agent as a Tool), callbacks and observability with ADK's built-in LoggingPlugin, and session, state, and memory management for reliable agent teams. The workshop is structured around ten progressive Jupyter notebooks and concludes with a capstone project: a full-stack Content Creation Studio that automates blog posts, social media content, email newsletters, and SEO metadata generation. Participants will also learn how to deploy AI agent systems to Google Cloud, using Agent Engine for agents and Cloud Run with a React and FastAPI stack for the frontend and backend.

Who is this workshop for:
Developers, AI engineers, cloud practitioners, and technical professionals who want a practical introduction to building and deploying multi-agent systems.

Key takeaways:
- Build and orchestrate multi-agent AI systems with Google ADK
- Create custom tools and agent teams powered by Gemini models
- Choose between LLM-driven and workflow-driven design patterns (Sequential, Parallel, Iterative)
- Navigate agent architecture trade-offs: Sub-agents vs. Agent as a Tool
- Implement callbacks, observability, and guardrails for production readiness
- Master session management, state, artifacts, and short-term vs. long-term memory
- Deploy production-ready AI systems on Google Cloud (Agent Engine + Cloud Run)
